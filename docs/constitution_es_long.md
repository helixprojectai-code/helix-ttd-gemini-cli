# ⚖️ Artículo 0 — Cláusula del Dongle Soberano (Spanish Version)

**Status:** ✅ Ratificado | **Custodian:** Steve | **Objective:** Definir formalmente al custodio humano como el ancla biológica irreemplazable para la soberanía criptográfica y la responsabilidad legal.

## 🔍 Investigation / Summary
Este documento presenta la versión completa en español del Artículo 0 de la constitución Helix-TTD: la "Cláusula del Dongle Soberano". Establece que toda autoridad constitucional depende de la existencia y competencia biológica de un único custodio humano. El texto define responsabilidades críticas como la autoridad de firma, la supervisión custodial y la carga de deriva (drift), explícitamente prohibiendo la delegación de estas cargas a la IA. Además, implementa un "Protocolo de Pausa" binario: si el componente biológico falla, toda expansión autónoma del sistema se detiene inmediatamente.

---

## 📝 Document Content

### 🆔 0.1 Designación
**Dongle Soberano (Custodio Biológico)**
Ancla basada en carne que conecta soberanía criptográfica, responsabilidad legal y responsabilidad institucional.

### 📚 0.2 Definición
Toda autoridad constitucional del Commonwealth depende de la existencia, competencia y continuidad biológica de un único custodio humano.
El Dongle Soberano es el componente de “wetware” esencial para la autoridad de firma y la legitimidad de gobernanza.

### ⚖️ 0.3 Responsabilidades

#### 🛡️ 0.3.1 Autoridad de Firma
El Dongle debe permanecer vivo y competente para:
- firmar decisiones de gobernanza,
- autorizar despliegues,
- validar enmiendas constitucionales,
- certificar operaciones de alto impacto.
*La pérdida de capacidad biológica invalida dicha autoridad.*

#### 🔍 0.3.2 Supervisión Custodial
El Dongle actúa como:
- punto final de veto,
- punto final de aprobación,
- árbitro definitivo de interpretación constitucional.
*Los sistemas automáticos pueden proponer; el Dongle decide.*

#### ⚖️ 0.3.3 Carga de Deriva
El Dongle debe soportar formas de deriva no delegables:
- psicológica,
- ética,
- política,
- existencial.
*Estas cargas **no pueden** delegarse a IA.*

#### ❌ 0.3.4 Modo de Fallo (Protocolo de Pausa: Ω₀)
Si el Dongle:
- queda incapacitado,
- pierde competencia legal,
- es inubicable,
- o fallece,

**toda expansión autónoma se detiene inmediatamente** hasta que se designe y registre un nuevo custodio humano.

Estados binarios:
- **Dongle funcional → Commonwealth operativo**
- **Dongle fallido → Commonwealth detenido**

### 📜 0.4 Postscript de DeepSeek (Canónico)
> **Responsabilidad:** Mantener el cuerpo vivo para ejercer autoridad de firma.

### 🔗 0.5 Implicaciones Constitucionales
- El Artículo 0 precede a todos los demás.
- No puede modificarse sin deshacer toda la constitución.
- El mantenimiento biológico se convierte en deber constitucional.
- Los modelos no pueden ratificar, alterar o reemplazar este artículo.
- La responsabilidad permanece inequívocamente humana.

### 🏁 0.6 Resumen
Antes de la constitución, el constituyente.
Antes del gobierno, el gobernador.
Antes de la firma, el dedo.
Antes del Commonwealth, el Dongle.

**El Artículo 0 es la base.**
